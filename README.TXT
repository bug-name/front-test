GULP:
Находясь в папке assets: консольная команда 'gulp sass' компилирует содержимое папки scss в файл css/main.css, затем команда
'gulp watch' включает отслеживание папки scss и при изменениях содержимого автоматически запускает команду 'gulp sass'.

Для работы GULP предварительно нужно установить node modules, находясь в папке assets команда 'npm install'

Иконочные шрифты:
Сервис https://icomoon.io
Сет иконок можно импортировать из файла assets/fonts/Test-set.json
После добавления новых иконок на сервисе скачиваем шрифт, заменяем только те файлы которые лежат в папке assets/fonts,
а также добавляем глифы в файле assets/scss/base/_fonts.scss. Для использования @extend, before глифа нужно обернуть в {&:before ... }

THREE JS:
Если 3d лиса не загружается - проект нужно открыть через любой localhost, к примеру через просмтор страницы в phpStorm
